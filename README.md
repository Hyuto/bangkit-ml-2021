## Run Exercise Notebooks on Locals

1. Install libraries<br>
   **Windows via PIP**

   ```
   py -m pip install -r requirements.txt
   ```

   **Linux via PIP**

   ```
   pip install -r requirements.txt
   ```
2. Run `prepare.py`<br>
   run `prepare.py` dan ikuti instruksinya

   **Windows**

   ```
   py prepare.py
   ```

   **Linux**

   ```
   python3 prepare.py
   ```
